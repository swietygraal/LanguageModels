# Language Models
Problem solutions from the Language Models course at University of Wrocław.

## Assignment 1
Chatbot from scratch (using one of the polish huggingface models), sentence probability, review classification (good/bad), zero-shot/one-shot/few-shot learning, heuristics for answering questions

## Assignment 2
Study on model abilities to perform arithmetic operations, study on gender/race/ethnicity bias, token generation methods, beam search, top-k and top-p filtering, token probability, text generation

## Assignment 3
Context-free and contextual token embeddings, transformer, BERT, ABX tests, review classification, data augmentation (using BERT/generative model/Word2Vec), logistic regression
